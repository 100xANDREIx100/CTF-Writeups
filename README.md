# 🚩 CTF Write-ups & Cybersecurity Portfolio

Welcome to my cybersecurity portfolio! This repository serves as a personal log of my journey through various Capture The Flag (CTF) challenges and wargames. 

My goal is not just to capture the flags, but to deeply understand the methodology, the tools used, and the underlying concepts learned along the way. I believe that documenting the process is just as important as finding the solution, but, above all, understanding why something works is the ultimate goal.

**Current Focus:** 
* 🐧 Linux Command Line Basics
* 🛡️ OverTheWire: Bandit
* 🎯 PicoCTF
